## Setup

```bash
$ git clone https://github.com/2gbeh/plateau-edu.git

$ cd plateau-edu

$ npm cache clean --force

$ npm install

$ npm run dev
```

## Usage

> Local: http://localhost:3000/

> Live: https://plateau-edu.vercel.app/
